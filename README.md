
# Minimal API Product Catalog

A lightweight RESTful API built with ASP.NET Core Minimal APIs and Entity Framework Core for managing a product catalog.

## 🏗️ Architecture

This solution follows a clean, modular architecture with separation of concerns:

### Projects Structure

- **Minimal-Api**: Main API project containing endpoints and application configuration
- **Minimal-Api-DbContext**: Data access layer with Entity Framework Core DbContext
- **Minimal-Api-DataClasses**: Domain models and data transfer objects

## 🚀 Features

- ✅ Full CRUD operations for products
- ✅ Entity Framework Core with SQL Server
- ✅ Minimal API endpoints with OpenAPI/Swagger documentation
- ✅ Async/await pattern for database operations
- ✅ RESTful API design
- ✅ Built on .NET 10

## 📋 Prerequisites

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)
- [SQL Server](https://www.microsoft.com/sql-server) (LocalDB, Express, or full version)
- [Visual Studio 2025](https://visualstudio.microsoft.com/) or [Visual Studio Code](https://code.visualstudio.com/)
